# Nordika Official Website

**Memorial website for Nordika - Synthpop & Electronic Music**

> In Loving Memory of Héctor Alejandro Marín Ríos (1968 - 2024)

## 🎵 About

Nordika is a synthpop/electronic music project from Mexico. This website serves as a memorial and archive for the complete musical legacy of the band.

## 🚀 Features

- **Bilingual**: Full support for Spanish and English
- **Responsive Design**: Optimized for all devices
- **Fast Performance**: Vanilla HTML/CSS/JS, no frameworks
- **SEO Optimized**: Schema.org, Open Graph, meta tags
- **Discography**: Complete album catalog with purchase options
- **Video Gallery**: YouTube video integration
- **Photo Gallery**: Lightbox image viewer
- **Contact**: WhatsApp integration for purchases

## 📁 Structure

```
├── index.html          # Main page
├── css/
│   └── styles.css      # All styles
├── js/
│   ├── main.js         # Main functionality
│   └── i18n.js         # Translation system
├── lang/
│   ├── es.json         # Spanish translations
│   └── en.json         # English translations
└── assets/
    └── images/         # All images
```

## 🖼️ Required Images

Add your images to these folders:

- `assets/images/hero/hero-bg.jpg` - Main hero background
- `assets/images/albums/*.jpg` - Album covers
- `assets/images/gallery/*.jpg` - Photo gallery images
- `assets/images/icons/favicon.png` - Site favicon

## 🔗 Configuration

Update these in `index.html`:

1. **WhatsApp Number**: Replace `XXXXXXXXXX` with actual number
2. **YouTube Video IDs**: Replace `VIDEO_ID_HERE`
3. **Social Media Links**: Update all `href="#"` with real URLs
4. **Bandcamp Links**: Add direct links to albums

## 🌐 Deployment (GitHub Pages)

1. Create a GitHub repository
2. Push this code to the repository
3. Go to Settings > Pages
4. Select "Deploy from a branch" > `main` > `/ (root)`
5. Your site will be live at `https://username.github.io/repo-name`
